## 🔗 与MetaSeekOJ集成

虽然完全独立，但也可以轻松集成：

### 集成方式1：事件触发
```javascript
// 在课堂页面触发
window.dispatchEvent(new CustomEvent('guardian-enable', {
  detail: { sessionId: 123, whitelist: ['metaseekoj.com'] }
}));
```

### 集成方式2：直接通信
```javascript
// 向插件发送消息
chrome.runtime.sendMessage(extensionId, {
  action: 'updateConfig',
  data: { enabled: true }
});
```
