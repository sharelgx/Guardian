## 步骤1️⃣：启动管理后台


### 或者手动启动：
```bash

python3 -m http.server 8888
```

## 步骤2️⃣：安装浏览器插件

### Chrome浏览器：

**①** 打开新标签页，地址栏输入：
```
chrome://extensions/
```
然后按回车

**②** 在右上角找到「**开发者模式**」，打开开关（变成蓝色）

**③** 点击左上角「**加载已解压的扩展程序**」按钮

**④** 在弹出的文件选择窗口中，找到并选择这个文件夹：
```
/home/sharelgx/MetaSeekOJdev/guardian-browser-extension/extension/
```

**⑤** 点击「选择」或「打开」

**✅ 成功！** 浏览器右上角工具栏会出现Guardian图标

### Edge浏览器：

**①** 地址栏输入：`edge://extensions/`
**②** 其他步骤完全一样

---

## 步骤3️⃣：开始使用

### 方式A：使用管理后台（推荐）

**①** 打开浏览器，访问：
```
http://localhost:8888
```

**②** 你会看到漂亮的管理界面：
- 左侧是功能菜单
- 右侧是内容区域

**③** 添加黑名单：
1. 点击左侧「🚫 黑名单」
2. 在输入框输入：`bilibili.com`
3. 点击「添加到黑名单」
4. 完成！

**④** 启用保护：
1. 点击左侧「📊 仪表盘」
2. 找到「启用保护」开关
3. 打开开关（变成绿色）
4. 完成！

**⑤** 测试：
- 访问 www.bilibili.com
- 会看到拦截页面！✅

### 方式B：使用浏览器插件（快速）

**①** 点击浏览器工具栏的Guardian图标

**②** 在弹出窗口中：
- 打开「启用保护」开关
- 点击「打开管理后台」

**③** 完成！

## 🎓 常用操作

### 快速拦截当前网站

```
1. 访问要拦截的网站（如B站）
2. 点击工具栏Guardian图标
3. 点击「加入黑名单」
4. 完成！下次访问就会被拦截
```

### 临时访问被拦截的网站

```
1. 访问被拦截的网站
2. 看到拦截页面
3. 点击「申请临时访问」
4. 输入管理员密码
5. 设置访问时长（如10分钟）
6. 确认，可以临时访问
```

### 关闭保护

```
方式1：点击工具栏图标 → 关闭开关
方式2：管理后台 → 仪表盘 → 关闭开关
方式3：拦截页面 → 点击「暂时禁用保护」
```

## ❓ 常见问题

### Q1：管理后台打不开？

**答**：确认服务器是否运行
```bash
# 重新启动
cd /home/sharelgx/MetaSeekOJdev/guardian-browser-extension
./start.sh
```

### Q2：插件安装了但没图标？

**答**：
1. 刷新扩展页面：`chrome://extensions/`
2. 确认插件已启用（开关是打开的）
3. 重启浏览器

### Q3：添加黑名单后还能访问？

**答**：检查三点
- ✓ 保护开关已打开？
- ✓ 域名格式正确？（如：bilibili.com，不要加http://）
- ✓ 是黑名单模式，不是白名单模式？

### Q4：怎么看孩子访问了什么网站？

**答**：
```
管理后台 → 访问日志 → 查看所有记录
```

### Q5：想限制每天上网2小时？

**答**：
```
管理后台 → 时间管理
→ 开启「每日时长限制」
→ 设置 120 分钟
→ 保存
```
